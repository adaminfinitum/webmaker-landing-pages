Welcome to the Lights On Afterschool Webmaker Challenge!

Start learning to code by making a digital poster

It’s fun and easy. This page shows you how to create and share your own digital poster. Make a poster showing what you love about your afterschool program as part of the Lights on Afterschool Webmaker Challenge!

You’ll learn a little HTML, the language that makes up the web, along the way. It’s perfect for beginners and experts alike and only takes a few minutes! Thimble, Mozilla’s code editor, makes it simple.
